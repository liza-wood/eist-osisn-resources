# eist-osisn-resources

Repository for Wood and Lubell (2025), "Resource constellations and institutional logics shape network structures of the organic seed niche innovation system" in _Environmental Innovation and Societal Transitions_.

Code for the paper's analysis is stored primarily in `code/paper_analysis.R`. For one section, this script points to the `challenges_expertise.R`.  Code for the analyses in the Appendix are stored primarily in `code/appendix/compiled_appendix.R`. At times this file sources material from other scripts, including `sample_representativeness.R` and `comparing_ergms.R`.
